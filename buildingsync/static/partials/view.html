Overview

This tool is used to take the entire relevant portion of the BuildingSync schema, currently at Version 2.0.0, and allow down-selection of the attributes for a given specific use case.

Schema "{$ schema_nickname $}"

The checkbox cells in the schema grid below include three checkboxes: Req (required), Opt (optional), and Ign (ignored).

  • Required means an instance file is invalid if it does not contain this attribute.
  • Optional indicates the attribute may be interpreted for this use case, but not necessary.
  • Ignored means this will not be checked on an instance file.
