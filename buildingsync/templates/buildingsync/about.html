About this tool

This tool is intended to tackle several tasks related to building sync files:

  • Parse and present the BuildingSync Schema
  • Allow users to specify different use cases, each of which include different parts of the BuildingSync Schema
  • Provide validation of BuildingSync instance files against the raw schema as well as specific use cases
  • Provide an API to allow other tools to get information about a specific use case or validity of a BuildingSync file
